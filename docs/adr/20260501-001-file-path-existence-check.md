# トラック検出後のファイルパス存在チェック

## Status

Accepted

## Context

Swinsian から取得したトラック情報にはファイルパスが含まれる。しかし、ファイルが移動・削除されている場合やパスが不正な場合、そのままドラッグ処理に進んでも djay Pro へのロードは失敗する。失敗時にも成功通知が出る問題もあり、早期にエラーとして検出したい。

一方、Music.app からはファイルパスが取得できないケース（空文字）もあるため、一律にチェックするとそのケースが使えなくなる。

## Decision

`main.rs` のトラック取得後、ファイルパスが空でない場合のみ `std::path::Path::exists()` でファイルの存在を確認する。存在しない場合は `anyhow::bail!` でエラー終了する。ファイルパスが空の場合はチェックをスキップする。

## Consequences

- 無効なファイルパスのままドラッグ処理に進むことを防止できる
- ファイルパスが空のケース（Music.app）は従来通り動作する
- ネットワークドライブ等でファイルアクセスが遅い場合、存在チェックで遅延が発生する可能性がある（実用上は問題ない）
