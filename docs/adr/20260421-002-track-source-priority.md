# トラックソースの優先順位

## Status

Accepted

## Context

Swinsian と Music.app の両方が同時に起動している場合、どちらからトラック情報を取得するかを決める必要がある。

## Decision

以下の優先順位でトラックを取得する:

1. アクティブ（フォアグラウンド）な Swinsian で選択中トラックあり
2. アクティブな Music.app で選択中トラックあり
3. 非アクティブな Swinsian で選択中トラックあり
4. 非アクティブな Music.app で選択中トラックあり

「アクティブなアプリを優先」を第一基準とし、同条件なら Swinsian を優先する。

## Consequences

- ユーザーが意図したアプリから取得される確率が高い（直前に操作していたアプリがアクティブなため）
- 両方非アクティブの場合は Swinsian が常に優先されるため、Music.app を主に使うユーザーには不便な場合がある
- 将来的に設定で優先順位を変更可能にする余地を残す
