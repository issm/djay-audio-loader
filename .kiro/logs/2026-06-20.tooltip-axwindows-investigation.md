# ツールチップ表示時の AXWindows 調査ログ

## 関連 Issue

- #13: Swinsianの楽曲テーブル行をマウスオーバーしてツールチップ表示中にホットキー入力すると楽曲選択に失敗する

## 目的

ツールチップ表示中にホットキーを押した際、楽曲の選択（AXSelectedRows の取得）に失敗する原因を調査する。

---

## 調査方法

`track.rs` の `get_track_from_swinsian()` にデバッグログを追加し、`RUST_LOG=debug` で実行。

追加したログ:
- `AXWindows` の数と各ウィンドウの role / subrole / title
- `find_all` で見つかった `AXTable` の数
- 各テーブルの `AXSelectedRows` の状態
- テーブルが見つからない場合、ウィンドウ配下のツリー構造（depth=4）

---

## 結果

### 正常時（ツールチップ非表示）

```
=== Swinsian AXWindows 数: 1 ===
  Window[0]: role=AXWindow subrole=AXStandardWindow title="Swinsian"
```

`AXWindows[0]` がメインウィンドウであり、テーブル探索が正常に動作する。

### 異常時（ツールチップ表示中）

```
=== Swinsian AXWindows 数: 2 ===
  Window[0]: role=AXHelpTag subrole= title=""
  Window[1]: role=AXWindow subrole=AXStandardWindow title="Swinsian"
=== AXTable 数: 0 ===
テーブルが見つからないため、ウィンドウのツリーをダンプします (depth=4):
role=AXHelpTag
```

---

## 原因

1. ツールチップは macOS の Accessibility API 上で `AXHelpTag` という role を持つ要素として `AXWindows` 配列に含まれる
2. ツールチップが表示されると、`AXWindows` 配列の**先頭（index 0）**に `AXHelpTag` が挿入される
3. 現行コードは `windows.first()` で最初のウィンドウを取得し、そこから `AXTable` を再帰探索している
4. `AXHelpTag` の子には `AXTable` が存在しないため、テーブルが見つからず `None` を返す → エラー

---

## 修正方針

`get_track_from_swinsian()` で `AXWindows` から取得するウィンドウを選ぶ際、`AXRole == "AXWindow"` のものだけをフィルタし、`AXHelpTag` 等をスキップする。

```rust
// 修正イメージ
let window = windows.iter().find(|&&w| {
    ax_string(w, "AXRole").as_deref() == Some("AXWindow")
})?;
```

---

## 修正結果

### 変更内容

`track.rs` の `get_track_from_swinsian()` にて、`windows.first()` を `AXRole == "AXWindow"` でフィルタする形に変更:

```rust
// 変更前
let window = *windows.first()?;

// 変更後 (#13)
let window = *windows.iter().find(|&&w| {
    unsafe { ax_string(w, "AXRole") }.as_deref() == Some("AXWindow")
})?;
```

### 修正後のログ（ツールチップ表示中）

```
=== Swinsian AXWindows 数: 2 ===
  Window[0]: role=AXHelpTag subrole= title=""
  Window[1]: role=AXWindow subrole=AXStandardWindow title="Swinsian"
=== AXTable 数: 1 ===
  Table[0]: AXSelectedRows=1 行
トラック取得完了: [Swinsian] Original Don (Extended Mix) / /Volumes/MASHIRO_EXT/Music/...
ドラッグ完了: デッキ 1 にロードしました
```

ツールチップ表示中でも `AXHelpTag` をスキップし、正しく `AXWindow` からテーブルを探索して楽曲のロードに成功した。
