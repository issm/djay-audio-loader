# djay Pro CLI ロードツール 調査メモ

## 目的

コマンドライン引数でファイルパスとデッキ番号を指定し、djay Pro の特定デッキに楽曲をロードする CLI ツールの実装。

```
load-to-djay <file_path> <deck_number>
```

背景: Swinsian の楽曲を djay Pro にドラッグ&ドロップでロードする操作の自動化。

---

## アプリケーション情報

- パス: `/Applications/djay Pro.localized/djay Pro.app`
- プロセス名: `djay Pro`

---

## 調査済み手段と結果

### 1. AppleScript 辞書
```sh
sdef "/Applications/djay Pro.localized/djay Pro.app"
# → 空（辞書なし）。AppleScript での直接操作は不可。
```

### 2. URL Scheme
```sh
plutil -p ".../Info.plist" | grep CFBundleURL
# → "djay" と "djay-app" の2つが存在
# → ただし strings 解析の結果、OAuth コールバック用途のみ
#    楽曲ロードには使えない
```

### 3. AppleScript open コマンド
```applescript
tell application "djay Pro" to open POSIX file "/tmp/test.mp3"
# → タイムアウトエラー (-1712)。使用不可。
```

### 4. System Events / Accessibility API ← 採用方針
- djay Pro は OpenGL/Metal ベースの描画だが、アクセシビリティ要素は取得可能
- ウィンドウ・ボタン情報の取得に成功

## 実装方針

### ドラッグ&ドロップのシミュレーション

djay Pro の波形エリア（ドロップゾーン）の座標を特定し、
AppleScript の `System Events` でファイルをドラッグ&ドロップする。

```applescript
tell application "System Events"
  -- ファイルをドロップゾーン座標にドラッグ
  set theFile to POSIX file "/path/to/track.mp3"
  -- drag theFile to {x, y}  ← 要検証
end tell
```

または `cliclick` コマンド（Homebrew でインストール可能）を使う方法：
```sh
brew install cliclick
cliclick dd:/path/to/file  # ドラッグ操作
```

### 座標の動的取得

ウィンドウ位置・サイズは毎回取得し、デッキ番号からドロップ先座標を計算する。
（ウィンドウを移動・リサイズしても対応できるよう）
