# Swinsian 選択項目の検出 検証ログ

## 目的

Swinsian で現在選択中のトラック情報（特にファイルパス）を Accessibility API で取得できるか検証する。

---

## UI 構造

```
AXApplication (Swinsian)
└── AXWindow
    └── AXSplitGroup (_NS:71)
        ├── AXScrollArea (_NS:81)          ← サイドバー
        │   └── AXOutline (_NS:85)         ← プレイリスト一覧
        └── ...
            └── AXScrollArea
                └── AXTable (_NS:25)       ← トラックリスト ✅
```

- サイドバーは `AXOutline`、トラックリストは `AXTable`
- `AXOutline` の `AXSelectedRows` はサイドバーの選択（例: "Library"）を返す
- `AXTable` の `AXSelectedRows` がトラックの選択行を返す

## 結論

✅ **`AXTable` の `AXSelectedRows` → 子要素の `AXCell` → `AXDescription` または `AXValue`** でトラック情報が取得できる。

選択変更後に再実行しても正しく新しい選択行の情報が返ることを確認済み（La Lu Sugarin' → Sunday Morning）。

### ファイルパスの取得方法

セルのインデックスは表示カラムの並び順に依存するため、インデックス固定は避ける。
`AXDescription` の値がファイルパスらしい文字列（`/` で始まる）かどうかで判定するか、
カラムヘッダーとの対応を取る方法が堅牢。

### 次のステップ

- カラムヘッダー (`AXColumn`) と各セルの対応を確認し、"File Path" カラムを確実に特定する方法を検討
- 複数選択時の挙動を確認
- 選択行の `AXPosition` を使ったドラッグ元座標の取得が可能か確認
